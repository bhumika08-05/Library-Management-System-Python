# Library Management System

## Overview

This is a simple Library Management System developed using Python and Object-Oriented Programming (OOP) concepts. The project allows users to add books, view available books, issue books, and return books through a menu-driven interface.

## Features

* Add new books to the library
* Display all books with their current status
* Issue books to users
* Return issued books
* Menu-driven console interface
* Object-Oriented Design

## Technologies Used

* Python 3
* Object-Oriented Programming (OOP)

## OOP Concepts Implemented

* Classes and Objects
* Encapsulation
* Constructors (`__init__`)
* Object Interaction

## Project Structure

* `Book` Class

  * Stores book title, author, and issue status.

* `Library` Class

  * Manages the collection of books.
  * Handles adding, displaying, issuing, and returning books.

## How to Run

1. Make sure Python is installed on your system.
2. Download or clone the repository.
3. Open a terminal in the project folder.
4. Run the following command:

```bash
python library_management.py
```
